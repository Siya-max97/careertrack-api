# CareerTrack API

## Features

- CRUD Job Applications
- PostgreSQL
- Prisma ORM
- Express
- Swagger Documentation
- Jest Testing

## Installation

npm install

## Run

npm run dev

## Test

npm test

## API

GET /
GET /health

GET /api/applications

GET /api/applications/:id

POST /api/applications

PUT /api/applications/:id

DELETE /api/applications/:id

Swagger

<http://localhost:3000/docs>
